# Análise de fatores que impactam o desempenho estudantil 

Este é um projeto desenvolvido na disciplina de Organização de Dados e consiste em uma **análise exploratória de dados (EDA)** sobre fatores que influenciam o desempenho acadêmico de estudantes, 
utilizando um dataset obtido do Kaggle. O dataset utilizado está disponível publicamente no link https://www.kaggle.com/datasets/lainguyn123/student-performance-factors.

## Arquivos do Projeto
- [`trabalhofinal_orgdados.ipynb`](trabalhofinal_orgdados.ipynb) | Notebook com a análise exploratória sobre o dataset escolhido
- [`TrabalhoFinalOrgDados.py`](TrabalhoFinalOrgDados.py) | Aplicação usando Streamlit para visualizar os resultados de modo interativo
- [`StudentPerformanceFactors.csv`](StudentPerformanceFactors.csv) | Dataset utilizado em formato de CSV
